Symphony
of
Death

Symphony of Death is a collection of experiments
translating Halo 5 match events into
interactive pages using WebGL and Web Audio.

Source Halo 5 Api

Interactive

Maps out each kill of a match as a vibrating string in 3D space. The resulting instrument is like a harp.

Suggestions and contributions are always welcome.
